Projecte web bàsic sobre pàdel.

Fitxers principals:
- index.html
- info.html
- multimedia.html
- mapa.html
- contacte.html
- css/styles.css
- js/script.js
- images/

Com obrir-ho:
1. Descomprimeix el ZIP.
2. Obre la carpeta a Visual Studio Code.
3. Fes doble clic a index.html o usa Live Server si el tens instal·lat.

El projecte compleix els criteris bàsics del PDF:
- 5 pàgines mínim
- layouts diferents amb CSS Grid
- header i footer comuns
- pàgina multimèdia amb imatges de pàdel
- pàgina amb iframe de Google Maps
- pàgina de contacte amb formulari
- disseny responsive amb 3 breakpoints
- fitxers separats en carpetes

Contingut multimèdia afegit:
- 4 imatges locals relacionades amb el pàdel dins la carpeta images
- 1 vídeo incrustat de YouTube sobre normes bàsiques del pàdel
